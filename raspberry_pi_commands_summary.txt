
Raspberry Pi Useful Commands Summary
====================================

ðŸ”¹ Initial Setup
----------------
Find Raspberry Pi Model:
    cat /sys/firmware/devicetree/base/model

Get Total and Free Memory:
    free -h

SD card Storage :
    df -h

Check OS Version:
    cat /etc/os-release

Check OS Architecture (32/64-bit):
    uname -m

ðŸ”¹ Software Update Commands
---------------------------
Update Package List:
    sudo apt update

Full System Upgrade:
    sudo apt full-upgrade

Remove Unneeded Packages:
    sudo apt autoremove

Clean Up Repository:
    sudo apt-get clean

Exit Terminal:
    exit

ðŸ”¹ Restart and Config
---------------------
Reboot Raspberry Pi:
    sudo reboot

System Configuration Tool:
    sudo raspi-config

ðŸ”¹ Hostname Configuration
-------------------------
View Hostname:
    hostname

Change Hostname:
    sudo nano /etc/hostname
    sudo nano /etc/hosts

ðŸ”¹ Text Editing (Nano)
----------------------
Edit File:
    sudo nano [file path]

Useful Log Files:
    sudo nano /var/log/messages
    sudo nano /var/log/syslog
    sudo nano /var/log/dmesg

ðŸ”¹ Power and Temperature
------------------------
Check Pi Temperature:
    /usr/bin/vcgencmd measure_temp

ðŸ”¹ EEPROM Update (Pi 4 Only)
----------------------------
Check EEPROM Update:
    sudo rpi-eeprom-update

Apply EEPROM Update:
    sudo rpi-eeprom-update -a
    sudo reboot
